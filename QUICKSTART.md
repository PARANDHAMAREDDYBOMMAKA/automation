# Quick Start - 3 Steps Only!

## Step 1: Push to GitHub

```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/kalvium-worklog-automation.git
git push -u origin main
```

## Step 2: Deploy to Render.com

1. Go to render.com
2. Click **New + → Blueprint**
3. Connect GitHub repo
4. Click **Apply**
5. Wait 5 minutes for deployment

## Step 3: Configure Cookies

1. Open your app URL from Render
2. Get cookies from kalvium.community (F12 → Application → Cookies)
3. Paste in form
4. Click Save

## Done!

Runs automatically every weekday at 5:00 PM IST.
